# OJCollegeEA
教务系统登录助手

现阶段功能：

1、查询历史学期成绩

2、查询课程表和课程信息

3、一键五星评教

适配进度：

温州大学瓯江学院（已适配）

温州大学（适配中）

如有学校或学院教务系统为正方教务系统的可以联系作者进行适配
